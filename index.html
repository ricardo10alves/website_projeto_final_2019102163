Aplicação Web com Chatbot para o ISPGaya



Este projeto consistiu na criação de uma aplicação web com um chatbot para o ISPGaya. Foram utilizadas técnicas de web scraping que extraíram os dados do site do ISPGaya e os armazenaram num ficheiro JSON. O chatbot foi programado para responder a perguntas de utilizadores com base nesses dados, melhorando a acessibilidade e a experincia de procura de informações sobre a instituição.
